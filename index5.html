Intro Hidro Homo Texere

The generation of fake or non-existant faces through the using of GANs have popularized in the recent years, especially through projects as the one hosted in the website https://thispersondoesnotexist.com/. This website, indeed, reflects the bias of accesible data sets, faces generated tend to reflect people with certain fenotypical characteristics, alongside a similar fashion and make-up, if any: that of populations inhabiting the Western Hemisphere. For the Singing Rivers project I have put together a data set of inhabitants of the Peruvian Rainforest, and trained a pre-trained Stylegan3 with this data, so to see how the machine learns and process these differences. Facial decorations, hair dresses, and the differences in features are not an easy task for the GAN to process, even if the pre-trained model was based on faces of people.

As all the previous models used in this project, this Stylegan3 model will continue to be trained upon new found data.
